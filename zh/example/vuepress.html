# 在 Vuepress 中使用

根据版本的不同，有两种方式引入。

# 0.x

module.exports = {
  markdown: {
    config: (md) => {
      md.use(require('markdown-it-vuese'), { /* options */ })
    }
  }
}

详情参阅 markdown.config

# 1.x

module.exports = {
  markdown: {
    extendMarkdown: (md) => {
      md.use(require('markdown-it-vuese'), { /* options */ })
    }
  }
}

详情参阅 markdown.extendMarkdown
