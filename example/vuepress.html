# Using in Vuepress

According to the different versions, there are two ways.

# 0.x

module.exports = {
  markdown: {
    config: (md) => {
      md.use(require('markdown-it-vuese'), { /* options */ })
    }
  }
}

For details markdown.config

# 1.x

module.exports = {
  markdown: {
    extendMarkdown: (md) => {
      md.use(require('markdown-it-vuese'), { /* options */ })
    }
  }
}

For details markdown.extendMarkdown
